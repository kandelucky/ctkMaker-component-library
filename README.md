# CTkMaker Hub

Community library of reusable components (`.ctkcomp`) for [CTkMaker](https://github.com/kandelucky/ctk_maker).

**Browse the library:** [kandelucky.github.io/ctkMaker-component-library](https://kandelucky.github.io/ctkMaker-component-library)

## What is a component?

A `.ctkcomp` file is a self-contained widget bundle saved from CTkMaker — buttons, cards, layouts, forms, full screens. Drop one into a project and it appears in your Components panel, ready to drag onto the canvas.

## Want to share one?

See [CONTRIBUTING.md](CONTRIBUTING.md). Short version: fork, drop your `.ctkcomp` and a preview `.png` into `components/`, add an entry to `index.json`, open a PR.

## Categories

`buttons` · `inputs` · `display` · `layout` · `navigation` · `feedback` · `templates`
